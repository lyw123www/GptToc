# ChatGPT 提问目录插件

一个用于 ChatGPT 网页端的右侧提问目录插件。插件会自动识别当前对话中的用户提问，生成可点击目录，方便在长对话中快速定位、回看和整理内容。

![插件预览](icons/image.png)

## 功能特点

- **自动生成提问目录**：在 ChatGPT 页面右侧展示当前对话里的用户提问列表。
- **点击快速跳转**：点击目录项即可滚动定位到对应的用户提问。
- **子标题展开**：如果回答内容中包含标题，会在对应提问下展示可展开的子标题目录。
- **颜色标记**：右键点击目录标题，可以选择颜色标记，也可以清除颜色。
- **标记持久保存**：颜色标记会按对话保存；切换对话后再回来，标记依然保留。
- **本地备份标记**：颜色标记会同步备份到 `chatgpt.com` 的本地 `localStorage`，删除并重新安装扩展后仍有机会恢复。
- **导出 Markdown**：支持将当前 ChatGPT 对话导出为 `.md` 文档，方便归档、整理和二次编辑。
- **可折叠侧栏**：目录面板支持收起，减少对页面的遮挡。
- **可拖拽调宽**：可以拖拽面板左侧边缘调整目录宽度。

## 安装方式

1. 打开 Chrome，进入：

   ```text
   chrome://extensions/
   ```

2. 打开右上角 **开发者模式**。
3. 点击 **加载已解压的扩展程序**。
4. 选择本项目文件夹：

   ```text
   GptToc
   ```

5. 打开或刷新 `https://chatgpt.com/`，即可使用。

## 使用说明

### 跳转目录

- 右侧会自动显示当前对话的提问目录。
- 点击目录标题，可以快速跳转到对应提问。
- 点击目录右侧箭头，可以展开或收起该轮回答中的子标题。

### 颜色标记

- 在目录标题上点击右键。
- 选择一种颜色即可完成标记。
- 选择 **清除颜色** 可以移除标记。
- 标记按对话区分，不同对话之间不会串色。

### 导出 Markdown

- 点击目录标题栏中的 **导出** 按钮。
- 插件会整理当前对话内容并下载为 Markdown 文件。
- 导出的 `.md` 文件适合用于笔记整理、知识库归档或后续编辑。

## 数据保存说明

- 面板宽度、折叠状态、颜色标记会保存在浏览器本地。
- 颜色标记同时写入扩展存储和 `chatgpt.com` 的站点 `localStorage`。
- 如果清除了浏览器站点数据、缓存、Cookie，或更换浏览器 / Chrome 用户配置，颜色标记可能无法恢复。
